SDF Survival Core · Integrations
INTEGRATIONSDEPENDENCIES

Required and optional integrations for SDF Survival Core. Make sure required ones are installed before first launch.

Required Integrations — Apollo & Paper must be installed. The plugin will not load without them.
Required Integrations
🌙
Apollo
Lunar Client API
Required
The Lunar Client API used by SDF Survival Core to deliver enhanced player experiences — custom waypoints, team indicators, rich title overlays and Lunar-native HUD elements.
Rich Title API Waypoints Team Display Custom HUD Combat Tags
GitHub → LunarClient/Apollo
📄
Paper
1.21+ recommended
Required
SDF Survival Core is built specifically for Paper. It relies on Paper's extended API for async chunk loading, event system, performance optimizations and modern packet handling.
Paper API Async Chunks Event System MC 1.21 Performance
papermc.io/downloads
Optional Integrations
🔑
LuckPerms
Permission Manager
Optional
Enables rank-based permission nodes for all SDF systems. Without LuckPerms, permission management must be handled by your server's default system.
Rank Nodes Per-Group Config VIP Tiers Staff Roles
luckperms.net
💰
Vault
Economy API
Optional
Required only if you want to use economy features — balances, shops, bounties and economy commands. Without Vault the economy module will be automatically disabled.
Economy API Balances Bounty System Shop Integration
SpigotMC → Vault
📊
PlaceholderAPI
PAPI Variables
Optional
Exposes all SDF stats as PAPI placeholders usable across your server — scoreboards, tab lists, chat formats and any other PAPI-compatible plugin.
Custom Variables Scoreboard Tab List Chat Format
SpigotMC → PAPI
🛡
WorldGuard
Region Control
Optional
Integrates with SDF to allow region-based PvP control — safe zones, spawn protection, no-PvP areas and zone-specific rules for KOTH and combat systems.
PvP Regions Safe Zones Spawn Shield KOTH Zones
BukkitDev → WorldGuard
