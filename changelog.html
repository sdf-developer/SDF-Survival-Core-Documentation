Release history
v1.0.0
Latest
Initial Release
⚡ COMING SOON
The first public release of SDF Survival Core — a complete, all-in-one plugin built from the ground up for modern Survival servers running Minecraft 1.21. Every core system is included, fully documented and ready to configure.
Core Systems
  • KOTH — King of the Hill with scheduling, capture zones and rewards
  • Clans — Full management: alliances, bank, chat, leaderboards
  • Combat PvP — Combat tags, logout detection, damage indicators
  • Economy — Vault-integrated with balance, pay and eco commands
  • Crates — Animated loot boxes with key types and hologram displays
  • Bounties — Player-driven bounty system with auto-payout
Player Features
  • Homes & TPA — Multiple homes per rank, TPA requests and warmup timers
  • Leaderboards — Hologram displays and GUI menus for kills, deaths, playtime
  • Kits — Per-rank cooldown timers and GUI kit selection menu
  • AFK Zone — Auto-detect idle players with configurable rewards
  • Scoreboard — Animated, per-world with PlaceholderAPI support
  • Tabulator — Custom tab list, headers, footers and nickname system
Staff & Admin
  • Staff Mode — Full moderation suite with vanish, freeze and inspect
  • Social Spy — Monitor private messages in real time
  • Staff Chat — Private channel exclusive to the moderation team
  • Broadcast — Server-wide announcements with custom formatting
  • Chat Control — Mute, clear and delay commands for chat management
Performance
  • ClearLag — Configurable entity cleanup with countdown warnings
  • MobStack — Auto-stack nearby mobs to reduce entity count and improve TPS
  • MySQL Support — Full database integration for persistent storage
  • Lunar Client API — Enhanced integration for Lunar Client users
Integrations
  • Vault — Full economy API support
  • LuckPerms — Permission node management
  • PlaceholderAPI — Variables across all systems
  • WorldGuard — Region-based PvP and spawn control
Configuration
  • Auto-generated config.yml on first launch
  • Per-module enable/disable flags in config
  • Multi-language support: English, Spanish, French
  • All messages customizable in messages.yml
